ACM COLLECTIONS - Frontend (React + Vite)

How to run:
1. Extract the ZIP to a folder.
2. Open the folder in VS Code (File -> Open Folder).
3. Open Terminal in VS Code (Terminal -> New Terminal).
4. Run: npm install
5. Then: npm run dev
6. Open the shown local URL in your browser (e.g., http://localhost:5173)

Project structure:
- package.json           # scripts & dependencies
- vite.config.js
- tailwind.config.cjs
- postcss.config.cjs
- index.html
- src/
  - main.jsx
  - App.jsx
  - index.css

Notes:
- The project uses Tailwind CSS. If styling looks off, ensure you installed dependencies and run the dev server.
- To add real images, replace the placeholder divs (bg-zinc-800 blocks) with <img src="/path/to/image.jpg" /> and add images to the public/ folder.
